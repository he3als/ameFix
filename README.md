# ameFix
Disables Defender to fix an AME Wizard (0.7) issue with Defender not being disabled properly at the 'Preparing System' stage.

This is not meant to permenantly disable Defender, and is only intended as a quick fix to the AME Wizard issue. AME Wizard v0.7.1 will have this fixed.
